## 📋 О проекте

SkillPass - это современная образовательная платформа, где пользователи могут проходить структурированные пути обучения, получать опыт (XP), повышать уровень и делиться своими достижениями. Проект создан с упором на геймификацию и социальное взаимодействие.

### 🎯 Ключевые особенности

- **Структурированное обучение** - пути, этапы и шаги
- **Геймификация** - XP, уровни, достижения
- **Социальные профили** - публичные профили с достижениями
- **Таблица лидеров** - соревнуйся с другими студентами
- **Система тестов** - проверка знаний с проходным баллом
- **Сохраненные пути** - откладывай интересные курсы на потом
- **Поиск людей** - находи друзей и коллег

## 🛠 Технологический стек

### Backend
- **Python 3.10+** - основной язык
- **Flask 3.0** - веб-фреймворк
- **SQLAlchemy** - ORM для работы с БД
- **Flask-Login** - аутентификация
- **Flask-Admin** - админ-панель
- **Flask-WTF** - формы и валидация
- **Werkzeug** - безопасность паролей

### Frontend
- **Bootstrap 5** - стили и компоненты
- **Jinja2** - шаблонизатор
- **Bootstrap Icons** - иконки
- **JavaScript** - интерактивность

### База данных
- **SQLite** (разработка)
- **PostgreSQL** (продакшн)

## 🚀 Установка и запуск

### Предварительные требования
- Python 3.10 или выше
- pip (менеджер пакетов)
- virtualenv (рекомендуется)

### Пошаговая установка

1. **Клонируй репозиторий**
```bash
git clone https://github.com/yourusername/skillpass.git
cd skillpass
```

2. **Создай виртуальное окружение**
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

3. **Установи зависимости**
```bash
pip install -r requirements.txt
```

4. **Создай файл .env**
```env
FLASK_APP=run.py
FLASK_ENV=development
SECRET_KEY=your-super-secret-key-change-this
DATABASE_URL=sqlite:///app.db
```

5. **Инициализируй базу данных**
```bash
python init_db.py
```

6. **Заполни тестовыми данными (опционально)**
```bash
python seed_data.py
```

7. **Запусти приложение**
```bash
python run.py
```

8. **Открой в браузере**
```
http://localhost:5000
```

## 🎮 Геймификация

### Система уровней
```
Уровень = √XP + 1
```

### Награды
- **Теория** - 10-20 XP за шаг
- **Тесты** - 30-70 XP за успешное прохождение
- **Достижения** - за завершение путей
