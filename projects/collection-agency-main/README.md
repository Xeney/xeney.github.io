# 🏢 CRM для Коллекторского Агентства

**Персональный учебный проект**, созданный мной с нуля как важный этап моего роста как веб-разработчика. Это не просто CRUD-приложение — это полноценная система, решившая реальную бизнес-задачу и продемонстрировавшая мою способность проектировать, реализовывать и доводить до рабочего состояния сложные веб-приложения.

> 💡 **Цель проекта** — не только автоматизировать работу коллекторского агентства, но и **инвестировать в своё профессиональное развитие**: от архитектуры кода и безопасности до UX и деплоя.

---

## 🚀 Что реализовано

### 📋 Основной функционал
- **Полный CRUD для должников** — создание, просмотр, редактирование и удаление
- **История взаимодействий** — ведение записей о звонках, SMS, email и личных встречах
- **Импорт и экспорт в Excel** — массовая загрузка и выгрузка данных одним кликом
- **Умный поиск и фильтрация** — по ФИО, номеру договора или банку
- **Профессиональная пагинация** — по 25 записей на странице

### 👥 Гибкая система управления пользователями
- **Ролевой доступ**:
  - 👑 **Администратор** — полный контроль: управление пользователями, редактирование данных
  - 👔 **Менеджер** — работа с должниками, импорт/экспорт
  - 👨‍💼 **Оператор** — просмотр и фиксация взаимодействий

### 🌐 Современный и удобный интерфейс
- **Публичная страница** — с реквизитами агентства и ссылкой на Федресурс (в соответствии с ФЗ-230)
- **Полностью адаптивный дизайн** — идеально выглядит на любом устройстве
- **Профессиональный UI** — на основе Bootstrap 5 с кастомной стилизацией и анимациями

---

## 🛠 Технологический стек

- **Backend**: Python, Flask (2.3.3), SQLAlchemy, Flask-Migrate
- **Frontend**: Bootstrap 5, Jinja2, Font Awesome
- **База данных**: SQLite (для разработки), легко масштабируется до PostgreSQL
- **Безопасность**: Хеширование паролей, CSRF-защита, валидация всех входных данных
- **Инструменты**: openpyxl для работы с Excel, Flask-Login для аутентификации

## 📦 Быстрый старт

### 1. Установка зависимостей
```bash
# Создаем и активируем виртуальное окружение
python -m venv venv
venv\Scripts\activate    # Windows
# source venv/bin/activate  # Linux/Mac

# Устанавливаем зависимости
pip install -r requirements.txt
```

### 2. Настройка и запуск
```bash
# Копируем настройки
copy .env.example .env

# Инициализируем базу данных
flask db upgrade

# Запускаем приложение
python run.py
```

**Готово!** Откройте в браузере: `http://localhost:5000`

**Доступ администратора по умолчанию:**
- **Логин**: `admin`
- **Пароль**: `admin123`

> **Важно**: После первого входа обязательно смените пароль!
